# Lista-de-Eventos-Trabalho
# 📌 Sistema de Eventos - Projeto com Flexbox

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como atividade da disciplina de **Programação Web**, com o objetivo de praticar a organização de layouts utilizando **CSS Flexbox**.

A aplicação consiste em uma página simples de um sistema de eventos, contendo:

* Cabeçalho com menu de navegação
* Banner principal
* Área de filtros
* Lista de eventos em formato de cards
* Rodapé

---

## 🎯 Objetivo

O principal objetivo da atividade foi aplicar conceitos de **Flexbox** para estruturar e alinhar os elementos da página de forma organizada e responsiva.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox)

---

## 📂 Estrutura do Projeto

```
📁 projeto
 ├── listaevento.html
 └── style.css
```

---

## ⚙️ Funcionalidades

* Layout organizado com Flexbox
* Menu de navegação horizontal
* Banner centralizado
* Filtros organizados em coluna
* Cards de eventos alinhados lado a lado com quebra de linha
* Rodapé centralizado

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados conceitos importantes como:

* `display: flex`
* `justify-content`
* `align-items`
* `flex-direction`
* `flex-wrap`
* `gap`
